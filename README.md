# Machine Learning Projects

This repository contains a collection of beginner-friendly Machine Learning projects built with Python.  
Each project uses a dataset, performs data preprocessing, trains a Machine Learning model, evaluates the model accuracy, and makes predictions using new input data.

## Projects

### 1. Diabetes Prediction

This project predicts whether a person is diabetic or not based on medical information.

**Features used:**

- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age

**Algorithm used:**

- Support Vector Machine (SVM) with linear kernel

**Accuracy:**

- Training Accuracy: 78.66%
- Testing Accuracy: 77.27%

---

### 2. Fake News Detection

This project predicts whether a news article is real or fake using Natural Language Processing.

**Dataset columns:**

- id
- title
- author
- text
- label

**Text preprocessing steps:**

- Handling missing values
- Combining author and title
- Removing special characters
- Converting text to lowercase
- Removing stopwords
- Stemming
- TF-IDF vectorization

**Algorithm used:**

- Logistic Regression

**Accuracy:**

- Training Accuracy: 98.63%
- Testing Accuracy: 97.90%

---

### 3. Sonar Rock vs Mine Prediction

This project classifies sonar signals as either Rock or Mine.

**Classes:**

- R: Rock
- M: Mine

**Algorithm used:**

- Logistic Regression

**Accuracy:**

- Training Accuracy: 83.42%
- Testing Accuracy: 76.19%
